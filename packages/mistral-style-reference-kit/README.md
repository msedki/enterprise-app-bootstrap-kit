# Mistral Docs Style Reference Kit

Kit non officiel de référence visuelle, fondé sur les styles publics de la documentation Mistral au 21 juin 2026.

## Contenu

- tokens de couleurs en CSS et JSON ;
- palette claire et sombre ;
- typographie et échelle ;
- tailles, rayons, espacements et motion ;
- composants CSS autonomes ;
- pack de 24 icônes pixel originales ;
- composants React d’exemple ;
- preview HTML et PNG ;
- guide de style détaillé ;
- carte des sources et notices de licence.

## Aperçu

Ouvre :

```txt
preview/index.html
```

ou consulte :

```txt
preview/preview.png
```

## Installation rapide

```html
<link rel="stylesheet" href="./tokens/theme.css">
<link rel="stylesheet" href="./components/components.css">
<body class="theme-mistral-reference msr-app">
```
